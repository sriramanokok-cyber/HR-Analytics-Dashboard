 HR Analytics Dashboard

 Overview
This project is an interactive HR Analytics Dashboard built using Power BI to analyze employee attrition patterns and workforce demographics.


Key Metrics
- Total Employees: 1470
- Attrition Rate: 16.12%
- Average Age: 36.92
- Average Monthly Income: 6.50K

 Features
- Department-wise Attrition Analysis
- Gender Distribution
- Overtime Analysis
- Age Distribution
- Job Role Attrition Analysis
- Interactive Filters:
  - Department
  - Gender
  - Job Role

 Tools Used
- Power BI
- DAX
- Data Modeling

Dataset
IBM HR Analytics Employee Attrition Dataset
